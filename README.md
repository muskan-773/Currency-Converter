# Currency-Converter
A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. This project allows users to convert currencies in real-time and automatically updates country flags based on the selected currencies.

## Features

* Convert currencies in real time
* Dynamic currency dropdown options
* Country flags update automatically
* Default conversion: USD → INR
* Input validation
* Responsive UI design
* Uses live exchange rate API

## Technologies Used

* HTML
* CSS
* JavaScript
* Currency API
* Flag CDN

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/CurrencyConverter.git
```

2. Open the project folder in VS Code.

3. Install the Live Server extension.

4. Right-click `index.html`.

5. Click **Open with Live Server**.
